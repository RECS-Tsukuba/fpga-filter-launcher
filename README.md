## フィルタのランチャ

### 概要
筑波大学工学システム学類の授業、知的工学システム応用実験及び機能工学システム
応用実験で行われるハードウェア実装実験で使用されるGUIランチャです。

### 使い方
```
./launcher.py
```
または
```
python3 launcher.py
```
を実行してください。

必要項目を入力後「開始」ボタンを押すと、フィルタリングを開始します。

### 必要環境
- python3
- python3-tk
